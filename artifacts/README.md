# Artifacts

This folder keeps only lightweight final result evidence.

Included:

```text
artifacts/publication_challenge_improved/publication_challenge_summary.json
artifacts/publication_challenge_improved/observation_metrics_compiled.json
artifacts/publication_challenge_improved/graphs/
```

Not included:

```text
per-seed checkpoints
per-seed prediction CSVs
ensemble prediction dumps
large generated run folders
```

Use this ignored folder for new local runs:

```text
artifacts/runs/
```
