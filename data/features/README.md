# Local WavLM Features

Put WavLM `.npy` feature files here when reproducing training or inference.

Expected final layout:

```text
data/features/WavLM_embeddings_unified/
├── en/
│   ├── fake/
│   └── real/
└── hi/
    ├── fake/
    └── real/
```

These files are intentionally ignored by Git because they are large generated feature artifacts.
