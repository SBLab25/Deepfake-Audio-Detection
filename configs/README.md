# Configs

This folder contains lightweight experiment configuration files.

Current file:

```text
safari_wavlm_auc96.yaml
```

It points to:

```text
src/safari_wavlm_ensemble_train.py
```

and uses the RawCh split CSVs under:

```text
data/splits/rawch/
```

The actual WavLM `.npy` feature files are not committed. Put them locally under:

```text
data/features/WavLM_embeddings_unified/
```
